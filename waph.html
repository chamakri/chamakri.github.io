    About the Course

    The "Web Application Programming and Hacking (WAPH)" course is designed to provide students with a comprehensive understanding of web application development and security. Students will study basic web application development with front-end technologies such as HTML5, CSS, and JavaScript, as well as back-end technologies like PHP and MySQL. The course also covers web application vulnerabilities and attacks, with hands-on exercises to explore these topics. Secure programming principles and practices are be introduced to avoid potential web application vulnerabilities and attacks.

    The course covers topics such as the Web and Web Application, Client-side Web Development with HTML5, CSS, and JavaScript, Server-side Web Application Development with PHP and Database (MySQL), Version control (git) and DevOps, Web Application Security: Vulnerabilities and Attacks with Hands-on Exercises, and Secure Web Programming Principles and Practices.

    Upon successful completion of the course, students will have the ability to effectively use state-of-the-art techniques in web application development and security tools for the analysis, design, and implementation of secure web systems. This will also enhance students programming skills to develop secure, robust, sustainable software products, and establish security analysis skills to identify and avoid potential web vulnerabilities. Will be able to apply practical knowledge of security in designing and implementing web applications with contemporary programming technologies.
